1. Apa itu Version Control?

Version Control merupakan sistem yang digunakan untuk mengelola perubahan pada file atau kode program. 2. Perbedaan Git dan GitHub:

Git adalah sistem version control yang digunakan untuk mengatur dan melacak perubahan kode secara lokal di komputer.

GitHub adalah platform web yang digunakan untuk menyimpan repository Git secara online sehingga bisa diakses dan dikerjakan bersama oleh banyak orang.

3. Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web dalam penyusunan web HTML dimisalkan sebagai body(isi) dari rumah.
